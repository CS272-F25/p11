About CoHabit

Our Mission

CoHabit is a mobile roommate-management platform designed to streamline shared living by centralizing communication, responsibilities, and financial coordination. Many households rely on ad-hoc systems—group chats, spreadsheets, verbal reminders—that are inefficient and prone to misunderstanding. CoHabit solves this by providing a unified, intuitive interface that helps roommates stay organized, accountable, and connected.

Why CoHabit?

The app's core features include a Roommate Directory for adding, removing, and managing household members and a Profile System where each user can customize their preferences, contact information, and living habits. CoHabit also integrates smart Notifications for chore reminders, payment deadlines, roommate requests, and schedule updates, reducing the cognitive load of daily coordination.

To support equitable cost-sharing, CoHabit offers a dedicated Finance Module where users can log expenses, split costs, track outstanding balances, and view historical transactions. This module minimizes financial ambiguity and encourages transparency across the household.

Accessibility & Design

We prioritize accessibility in every aspect of CoHabit:

  • Semantic HTML: Proper heading hierarchy (h1→h2→h3) for screen readers
  • Labeled inputs: All form fields have associated labels
  • Alt text: Images include descriptive alternative text
  • Color contrast: WCAG AA compliant contrast ratios
  • Keyboard navigation: Enhanced focus states for keyboard users
  • ARIA: Live regions for dynamic content updates

Technology Stack

This draft website demonstrates CoHabit's features using:

  • CSS3: Custom styles with Bootstrap 5 framework
  • JavaScript: Interactive features with localStorage for demo data persistence
